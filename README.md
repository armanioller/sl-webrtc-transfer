# SL WebRTC File Transfer

Este projeto é uma HUD para Second Life que permite a transferência de arquivos P2P diretamente entre dois avatares usando WebRTC e MQTT (para sinalização).

## Como Usar
1. Coloque o script `hud.lsl` no seu prim de HUD.
2. Anexe a HUD na sua tela.
3. Insira o UUID do destino e envie o arquivo P2P!
